Autonomous IoT Accident Detection System

ESP32 · EMBEDDED C++ · GPS/GSM
The Problem
Two-wheeler incidents in remote areas suffer delayed emergency response because crash notification still depends on manual reporting.
The Architecture
Engineered a smart modular unit around an ESP32 microcontroller paired with an MPU6050 accelerometer/gyroscope, relaying coordinates over cellular networks via integrated GPS/GSM hardware.
The Hurdles & Debugging
Solved noise-filtering issues in raw gyroscope telemetry that triggered false alarms during sudden braking, by introducing mathematical threshold constraints directly in the firmware.
Impact
Dispatches emergency coordinates within 5 seconds of threshold collision detection. Initiated formal applications to pursue patent protection for the architecture.
